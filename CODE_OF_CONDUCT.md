# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

## Summary of expectations

- Be respectful and constructive. Critique code and ideas, not people.
- No harassment, discrimination, or personal attacks of any kind.
- Assume good faith; disagree openly and civilly.
- Respect maintainers' time and the project's stated scope decisions.

## Enforcement

While the project is in private development, conduct concerns go to the
project owner. Before the first public release this file will be finalized
with a named enforcement contact and process, per the Contributor Covenant's
enforcement guidelines.
